DSA Lessons (Russian & English / Русский и Английский)

This repository contains my notes, code examples, and exercises for learning Data Structures and Algorithms (DSA).
Все уроки ведутся на русском и английском языках, с пояснениями, примерами кода на JavaScript и Python, а также ссылками на задачи на LeetCode.


Usage / Использование

Each lesson folder contains / Каждая папка урока содержит:

Notes in Russian and English / Заметки на русском и английском

Sample code in Python / Примеры кода на Python

Optional links to videos or resources / Дополнительные ссылки на видео или ресурсы


Notes / Примечания

Lessons are intended for both beginners and more experienced programmers / Уроки подходят как для новичков, так и для опытных разработчиков.

You can use this repository as a personal study guide or reference / Репозиторий можно использовать как личное руководство или справочник.
