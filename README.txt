# DSA Lessons (Russian)

This repository contains my notes, code examples, and exercises for learning **Data Structures and Algorithms (DSA)**.  
All lessons are conducted **in Russian**, with explanations, sample code in JavaScript and Python, and links to tasks on LeetCode.

## Lessons

1. **Lesson 1 Introduction**  
   Overview of the course, goals, approach to learning, and project ideas.

2. **Lesson 2 IDE BigO Arrays HashTables**  
   Setting up the IDE, understanding Big O notation, arrays, HashTables, and sample tasks.

3. **Lesson 3 Arrays HashTables Searching Sorting**  
   Deep dive into arrays, HashTables, searching algorithms (binary search), and introduction to sorting.

4. **Lesson 4 Array Sorting TimSort**  
   Sorting algorithms, Bubble Sort, Insertion Sort, TimSort, and exercises on array sorting.

## Usage

- Each lesson folder contains:
  - Notes in Russian
  - Sample code in JS/Python
  - Optional links to videos or resources
- Exercises can be tested locally in your IDE or on [LeetCode](https://leetcode.com/).

## Notes

- Lessons are intended for both beginners and more experienced programmers.
- You can use this repository as a personal study guide or reference.
